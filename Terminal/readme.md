# SSH Login:

```
ssh Benutzername@Domain -p 123456
Bsp.:
ssh root@lyra.et-inf.fho-emden.de -p 14042
```

# Passwort ändern:

```
passwd
```

# Ordner erstellen:

```
mkdir Ordnername
```

# Node Prjekt erstellen:

```
npm init
```

# Datei editieren und erstellen(Editor):
```
nano Dateiname
```

# Node-Datei ausführen:
```
node Dateiname
Bsp:
node index.js
```
